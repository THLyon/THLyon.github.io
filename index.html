Lyon Shops

Lyon Shops is a fully functional mock Ecommerce website. The application allows one to sign up with either an email address and password or their Google account. The user can sign up and or sign in, navigate the website adding various clothing and headwear, make final changes in their cart and check out utilizing Stripes external API. Check out the Github repository.

Technologies used includes:

  • React: Hooks
  • Firebase (Database, Authentication)
  • GraphQL

Gimme

Gimme is a chrome extension pop up and webpage built for expediated access to the PGA Tour scorecards. The product allows you to view the current score card, course details, keep track of your favorite players, and view the PGA Twitter timeline in a fast and organized location. Don't worry about leaving your current work to check on the scores, just click the icon in your extension library and immediately see the current leaderboard. Check out the Github repository.

Technologies used includes:

  • React: Hooks
  • React Router
  • MongoDB
  • Webpack

Solid Structure

Solid Structure is an open source devtool for the front end library SolidJS. SolidStructure renders dependency graphs and component trees displaying components and signal relationships via the data visualization library D3.js in a chrome extension dev tool. To view the product, navigate to the product repository on Github, or head over to solidstructure.io.
